## English Version

### DESCRIPTION
This project is a fun and challenging general trivia game that will test your knowledge across three difficulty levels:
- **Facile** (Easy)
- **Moyen** (Intermediate)
- **Difficile** (Hard)

Each level features 5 unique questions, and you'll have a 30-second timer ticking down for each one. Think fast, answer smart, and prove you're not braindead!

At the end of the game, you'll receive your final score along with a special appreciation message based on your performance.

### INSTALLATION
Getting started is super simple:

1. Download the 4 project files
2. Open them in Visual Studio Code (or your favorite code editor)
3. Install the HTML, CSS, and JavaScript extensions (if needed)
4. Boom, you're ready to play!

### USAGE
The interface is intuitive and user-friendly. Here's how to play:

1. Select your preferred difficulty level
2. Read each question carefully
3. Click on the answer you think is correct
4. Race against the 30-second timer
5. Complete all 5 questions to see your final score and appreciation

**Pro tip:** The buttons are clearly labeled, so just tap the answer that seems right to you!
